# notes-util

## CLI Commands

```bash
# install dependencies
yarn

# serve with hot reload at localhost:8080
yarn dev

# build for production with minification
yarn build

# test the production build locally
yarn start

# run tests with jest and preact-render-spy
yarn test
```
